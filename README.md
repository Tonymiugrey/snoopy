# macOS 史努比屏幕保护

素材是从 tvOS 中提取的，开源了，视频素材没放到版本库，可以在 Release 总下载安装包，在里边的 Resouces 中就能找到。

应用没有签名，所以打开的时候会提示危险什么的，在系统设置-隐私和安全性-滚到最后，仍然打开。

这个屏保的史努比出现顺序还没有摸透，因为 bugOS 的原因，昨晚浪费了极长时间进行调试，累了，随便了。

后续有空会研究研究到底怎么播放的。
